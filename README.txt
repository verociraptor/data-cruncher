## General info
This project analyzes a given dataset from a csv file and designs and test ways to 
represent the payments data.
	
## Technologies
Project is created with:
* Java Version: 1.14
* Gradle Version: 6.5.1
	
## Setup and Test Cases
To run this project and its test cases, go into the project directory 
and open the command prompt:

```
$ cd full/path/to/project/directory
$ gradle test
```


* To view the test cases done, go into src/test/java/DataCruncherTest.java
